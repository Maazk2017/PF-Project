## 🏫 Campus Event Management System (CEMS)

A robust, console-based academic ERP application engineered in **C** to automate core administrative, faculty, and student workflows. Designed with a strict focus on modular architecture and procedural programming patterns, this system provides a highly efficient, lightweight alternative to resource-heavy database engines by leveraging optimized file system persistence.

The application implements a secure, role-based access control (RBAC) gateway that partitions system privileges into distinct administrative dashboards, grade books, and student portals. Under the hood, it features custom-structured data models, robust binary/text file processing (`File I/O`), stream validation logic to handle runtime input anomalies, and dynamic record-filtering algorithms to ensure low-latency data retrieval.
